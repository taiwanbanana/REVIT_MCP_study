# Revit MCP 代理執行原則 (Agent Principles)

## 核心限制
- **嚴格遵循指令**：請切記，不能執行使用者沒有要求的指令。
- **操作透明化**：所有對模型幾何的修改必須明確記錄。
- **原點定義**：在 RFA 族群中，原點應視為載入後的載入點（結構平面與基本牆之交點）。

## 部署與覆蓋規範
- **正確部署路徑**：Revit 載入路徑為 `Addins\[Version]\RevitMCP\RevitMCP.dll`（子資料夾）。嚴禁放置於根目錄。
- **覆蓋成功判定**：執行安裝腳本成功且無鎖定錯誤，即代表 Revit 已關閉。此時禁止檢查連線狀態。
- **生效機制**：DLL 覆蓋後必須完全重啟 Revit 才能更新記憶體中的執行邏輯。
