# 台灣建築法規摘要

本文件提供 AI 輔助建築法規檢討時的參考標準。

## 一、容積率規定

### 1.1 使用分區容積率上限

| 使用分區 | 容積率上限 | 備註 |
|---------|:---------:|------|
| 第一種住宅區 | 60% | 低密度 |
| 第二種住宅區 | 120% | 中低密度 |
| 第三種住宅區 | 225% | 中密度 |
| 第三之一種住宅區 | 300% | 中高密度 |
| 第四種住宅區 | 300% | 高密度 |
| 商業區 | 560~630% | 依地區而異 |
| 工業區 | 210% | 一般工業 |

### 1.2 面積計算規則

| 項目 | 計入容積 | 說明 |
|-----|:-------:|------|
| 一般室內空間 | 100% | 客廳、臥室、辦公室 |
| 陽台 | 50% | 有條件計入 |
| 露台 | 0% | 不計入 |
| 機房 | 0% | 電梯機房、空調機房 |
| 樓梯間 | 特殊 | 依法規計算 |
| 地下室 | 0% | 一般不計入 |
| 屋頂突出物 | 特殊 | 超過標準需計入 |

## 二、防火規定

### 2.1 建築物防火構造標準

| 樓層數 | 主要構造 | 外牆 | 樑柱 |
|:-----:|:--------:|:----:|:----:|
| 1~3層 | 一般構造 | 30分鐘 | 30分鐘 |
| 4~6層 | 耐火構造 | 1小時 | 1小時 |
| 7~14層 | 耐火構造 | 2小時 | 2小時 |
| 15層以上 | 耐火構造 | 3小時 | 3小時 |

### 2.2 防火區劃

| 區劃類型 | 面積限制 | 防火時效 |
|---------|:-------:|:--------:|
| 水平區劃 | 1,500㎡ | 1小時 |
| 垂直區劃 | 每3層 | 1小時 |
| 特殊區劃 | 視用途 | 2小時 |

### 2.3 材質防火等級

| 等級 | 說明 | 常見材料 |
|-----|------|---------|
| 不燃材料 | 完全不燃 | 混凝土、磚、石材、鋼 |
| 耐燃一級 | 高度耐燃 | 石膏板、矽酸�ite板 |
| 耐燃二級 | 一般耐燃 | 耐燃木材 |
| 耐燃三級 | 輕度耐燃 | 耐燃合板 |

## 三、避難規定

### 3.1 走廊寬度

| 用途 | 兩側有居室 | 其他 |
|-----|:---------:|:----:|
| 住宅 | 1.2m | 0.9m |
| 辦公 | 1.6m | 1.2m |
| 學校 | 2.4m | 1.8m |
| 醫院 | 2.4m | 2.1m |

### 3.2 樓梯寬度

| 用途 | 最小寬度 | 備註 |
|-----|:-------:|------|
| 住宅 | 0.75m | 淨寬 |
| 辦公 | 1.2m | 淨寬 |
| 公共建築 | 1.4m | 淨寬 |

### 3.3 逃生距離

| 建築類型 | 最大步行距離 | 備註 |
|---------|:----------:|------|
| 住宅 | 40m | 至樓梯 |
| 辦公 | 30m | 至樓梯 |
| 商場 | 30m | 至出口 |
| 有自動灑水 | +10m | 可增加 |

## 四、無障礙規定

### 4.1 坡道

| 項目 | 標準 |
|-----|------|
| 坡度 | ≤ 1:12 |
| 寬度 | ≥ 0.9m |
| 平台 | 每 9m 設置 |

### 4.2 電梯

| 項目 | 標準 |
|-----|------|
| 最小尺寸 | 1.1m × 1.4m |
| 門寬 | ≥ 0.9m |
| 呼叫按鈕高度 | 0.8~1.2m |
